# Stack

## **Stack이란?**

{{< hint info >}}
후입선출(LIFO, Last-In-First-Out)의 원칙을 따른다, 새로운 데이터는 스택의 맨위에 쌓이게 되고, 가장 최근에 쌓인 데이터부터 차례로 꺼낸다.    
스택은 주로 함수 호출과 관련된 정보를 저장하는데 사용된다. 함수가 반환되면 스택에서 해당함수의 정보를 제거하여 이전 함수로 돌아간다.
{{< /hint >}}
